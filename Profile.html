Projects

  • AR Menu

    This project was made using Unity, C++, and Vuforia AR. This was made to overcome food wastage and good hygiene after we just overcame Corona Virus. The idea is that a barcode would be placed in each table in a restaurant. The customer can scan the barcode using their phone. This would take them to a Menu Card. The customer can then select which food he would like to order. Once the customer clicks the desired food, an AR model would be presented to him which would be placed in place of the barcode. Thus the customer can see how quantity of food they are ordering.

  • Book Recommendation

    This project was made using SQL and Python(for GUI, using tkinter). This was a initially made for book management, but it later tunred to be a book recommendation for the user. For example, if you are using this application and you returned or issued your regularly through this application, it would store your data in the database. Now if you ask it for recommendation, it will see your latest returned book and recommend you five books from each category(author's other works, topic related, genre related, year of release, popularity,etc).
